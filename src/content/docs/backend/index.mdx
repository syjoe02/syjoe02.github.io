Core backend engineering concepts: how systems receive, process, and persist data reliably at scale.

## Topics

- **API Design** — REST, gRPC, GraphQL tradeoffs; versioning; pagination; error contracts
- **Database Patterns** — indexing, connection pooling, N+1, optimistic locking
- **HTTP Internals** — keep-alive, connection handling, HTTP/2 multiplexing
- **Caching Strategies** — read-through, write-behind, cache invalidation, CDN patterns
