Debugging is a skill. This section covers systematic approaches to diagnosing complex failures.

## Topics

- **Systematic Debugging** — rubber duck debugging, binary search through state, hypothesis-driven debugging
- **Memory Leaks** — heap profiling, GC pressure, reference cycles
- **Race Conditions** — thread sanitizers, happens-before, lock analysis
- **Performance Profiling** — CPU flamegraphs, latency percentiles, tail latency
