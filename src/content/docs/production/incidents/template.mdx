## Root Cause

Describe the underlying technical failure. Be specific.

```
Example:
A memory leak in the connection pool was introduced in commit abc123.
Under sustained load, the process OOM-killed after ~4 hours.
The restart loop masked the issue in staging (lower traffic).
```
