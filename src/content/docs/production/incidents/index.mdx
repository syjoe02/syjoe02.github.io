A structured catalog of production failures — real incidents analyzed for patterns and prevention.

Each entry follows a consistent format:
- **Summary** — impact and duration
- **Timeline** — chronological sequence of events
- **Root Cause** — the underlying technical failure
- **Mitigation** — how it was resolved
- **Prevention** — what changed afterward
- **Lessons Learned** — reusable takeaways

## Contributing an Incident

Use the [incident template](/production/incidents/template/) as a starting point.
Remove any identifying information before publishing.
