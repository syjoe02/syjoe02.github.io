## What is this?

Engineering Compendium is a personal knowledge base for technical writing on backend systems, distributed architecture, and production engineering.

## What to expect

Posts here cover backend systems and API design, distributed systems concepts, production incidents and retrospectives, AI infrastructure patterns, and architecture decision records.

New posts are tagged so you can filter by what interests you.

## Creating new posts

Use the **Keystatic admin** at `/keystatic` (in dev mode) to create new posts, or add a `.md` file directly to `src/content/blog/`.

Each post has a title, description, tags, publish date, and markdown content body.
